- runOCRTest.bat: runs the OCR engine on file order.png
- runPDFTest.bat: runs the PDF to XMl conversionon order.pdf

- guide.htm: user guide
- lib directory: third party libraries
- j4locr.jar: Main jar
- leptonlibd.dll and tess3Wrapper.dll: TesseractOCR engine DLLs
